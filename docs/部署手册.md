### 部署手册

#### 前端部署

##### &nbsp;	1.进入 frontend/ 目录。

##### &nbsp;	2.执行 npm install 安装依赖。

##### &nbsp;	3.执行 npm run build 构建项目。

##### &nbsp;	4.将构建后的文件部署到静态服务器。

##### 后端部署

##### &nbsp;	1.进入 backend/ 目录。

##### &nbsp;	2.创建 Python 虚拟环境并激活。

##### &nbsp;	3.执行 pip install -r requirements.txt 安装依赖。

##### &nbsp;	4.配置数据库连接（config.py）。

##### &nbsp;	5.运行 python run.py 启动后端服务。

##### 数据库部署

##### &nbsp;	1.进入 database/ 目录。

##### &nbsp;	2.执行 schema.sql 创建数据库表结构。

##### &nbsp;	3.执行 dummy\_data.sql 导入模拟数据。

##### Docker 部署

##### &nbsp;	1.进入项目根目录。

##### &nbsp;	2.执行 docker-compose up -d 启动所有服务。
